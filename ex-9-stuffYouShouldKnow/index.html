Stuff You Should Know

  1. There is a screenshot of the completed state of this exercise inside the images folder. Drag this to your desktop to use as a guide for the following steps.
  2. The text in this page currently sits against the left margin. This is known as inadequate text stand-off. Fix it by looking at the Bootstrap documentation here.
  3. There is a stylesheet file called styles.css in the root folder but line 9 shows where this stylesheet should be moved to. Do so and then see if you see a change on the page.
  4. Take a look at the styles.css file. Look for the first /* yo student */ comment and then PONDER why the rule on the next line is not yet changing this page. Make the change within this file (not the stylesheet) to make the rule affect this page.
  5. You should also know how to write an HTML comment. Look up how do this at this webpage, and then add an HTML comment before the h1 on this page explaining what causes it to have a purple color.
  6. You should know how to refer to a file inside a folder both from inside out and outside in: So, look at styles.css again and look for the second /* yo student */, and then figure out how to make the image called graph-paper.svg be the background of the <header> element.
  7. Now that you have the background image, you have to figure out how to set the padding on the correct element so that the <h1> element is not sitting directly against the left margin of the area with a background. Add a declaration to the rule already inside styles.css to make this happen.
